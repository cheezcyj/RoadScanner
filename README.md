## 프로젝트 개요
|팀/프로젝트명|RoadScanner|
|------|---------|
|팀원|김정민, 송세림, 오동훈, 이주영, 장태근, 차지욱, 차현경, 최유정|
|프로젝트 종류|자율주행 표지판 인식 사이트|
|기간|2023.07.05 ~ 2023.08.29|
|프로젝트 목표|1. 딥러닝 알고리즘을 활용하여 교통 표지판을 식별하고 분류하는 모델 개발<br>2. 사용자의 직관적 이해를 돕기 위해 인식 성능 및 결과 시각화<br>3. 사용자 피드백을 통한 모델 평가 및 성능 개선|
